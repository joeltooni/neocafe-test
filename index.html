📱 MacroDroid Setup Guide

1
Open MacroDroid → Add Macro

Tap the + button to create a new macro

2
Set Trigger: SMS Received

Triggers → SMS Received → From: Any number → Body contains:

You have received
3
Set Action: HTTP Request

Actions → HTTP Request → configure as follows:

Method: POST URL: https://YOUR-SITE.netlify.app/api/webhook Headers: Content-Type: application/json
4
Set Request Body

In the Body field, paste exactly:

{"sms": "[sms_body]", "secret": "YOUR_WEBHOOK_SECRET"}

MacroDroid replaces [sms_body] with the actual SMS text automatically

5
Samsung Battery Fix

Settings → Apps → MacroDroid → Battery → set to Unrestricted

Without this, Samsung One UI will kill MacroDroid in the background

6
Test it

Send a real MoMo payment to the vendor number — the SMS should appear in the Live Feed above within a few seconds
